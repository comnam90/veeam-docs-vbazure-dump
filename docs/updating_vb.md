In this article

Veeam Backup for Microsoft Azure allows you to check for new product versions and available software software package updates. It is recommended that you timely install available software package updates to avoid performance issues while working with the product. For example, timely installed security updates may help you prevent potential security issues and reduce the risk of compromising sensitive data.
