| Important |
| If your backup appliance is managed by a Veeam Backup & Replication server and you add a new backup repository using the Veeam Backup for Microsoft Azure Web UI, Veeam Backup for Microsoft Azure will not propagate these settings to the Veeam Backup & Replication server automatically. To discover new backup repositories created in the backup appliance, follow the instructions provided in section [Connecting to Existing Repositories](repository_add_console_existing.md). |

To add a new backup repository, do the following:

1. [Launch the Add Repository wizard](repository_ui_wizard.md).
2. [Specify a repository name and description](repository_ui_name.md).
3. [Configure repository settings](repository_ui_settings.md).
4. [Enable encryption for the backup repository](repository_ui_encryption.md).
5. [Configure load control settings for the backup repository](repository_ui_load.md).
6. [Finish working with the wizard](repository_ui_finish.md).
