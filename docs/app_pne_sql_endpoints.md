In this article

To allow Veeam Backup for Microsoft Azure access to the databases that you want to protect, you must create private endpoints for your SQL Server.

You must create a separate private endpoint for every VNet to which worker instances are connected. To create a private endpoint, complete the following steps:

1. [Launch the Create a private endpoint wizard](app_pne_sql_wizard.md).
2. [Configure private endpoint settings](app_pne_sql_endpoint.md).
3. [Specify resource settings](app_pne_sql_resource.md).
4. [Specify network settings](app_pne_sql_configuration.md).
5. [Specify DNS settings](app_pne_sql_dns.md).
6. [Assign tags](app_pne_sql_tags.md).
7. [Finish working with the wizard](app_pne_sql_finish.md).
