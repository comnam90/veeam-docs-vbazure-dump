# Performing VM Restore Using Web UI


Veeam Backup for Microsoft Azure offers the following restore options:

* [VM Restore](entire_vm_restore_ui.md) — restores an entire Azure VM.
* [Disk Restore](performing_disk_restore.md) — restores virtual disks attached to an Azure VM.
* [File-level Restore](performing_flr.md) — restores individual files and folders of an Azure VM.

You can restore Azure VM data to the most recent state or to any available restore point.
