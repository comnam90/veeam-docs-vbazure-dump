In this article

To create a backup policy, do the following:

1. [Launch the Add Cosmos DB Policy wizard](cosmos_db_backup_wizard.md).
2. [Specify a backup policy name and description](cosmos_db_backup_name.md).
3. [Configure backup source settings](cosmos_db_backup_source_settings.md).
4. [Configure backup target settings](cosmos_db_target_settings.md).
5. [Configure processing options](cosmos_db_processing_options.md).
6. [Create a schedule for the backup policy](cosmos_db_backup_policy_schedule.md).
7. [Review the estimated cost of protecting the selected Cosmos DB accounts and databases](cosmos_db_backup_cost.md).
8. [Specify automatic retry, health check and notification settings for the backup policy](cosmos_db_backup_retry_notifications.md).
9. [Finish working with the wizard](cosmos_db_backup_finish.md).
