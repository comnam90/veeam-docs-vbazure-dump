# Editing Service Accounts


To edit a service account, do the following:

1. [Launch the Edit Account wizard](service_account_edit_wizard.md).
2. [Update the account name and description](service_account_edit_name.md).
3. [Connect to the Microsoft Entra application with which the account is associated](service_account_edit_connect.md).
4. [Change the account scope](service_account_edit_scope.md).
5. [Update account roles](service_account_edit_roles.md).
6. [Check the required permissions](service_account_edit_check_permissions.md).
7. [Finish working with the wizard](service_account_edit_finish.md).
