In this article

To restore the configuration database of a backup appliance, do the following:

1. [Launch the Configuration Restore wizard](configuration_restore_console_wizard.md).
2. [Choose a backup file](configuration_restore_console_file.md).
3. [Review the backup file info](configuration_restore_console_info.md).
4. [Specify a decryption password](configuration_restore_console_password.md).
5. [Choose restore options](configuration_restore_console_options.md).
6. [Specify a user whose credentials will be used to connect to the appliance](configuration_restore_console_credentials.md).
7. [Wait for the restore process to complete](configuration_restore_console_progress.md).
8. [Finish working with the wizard](configuration_restore_console_finish.md).
