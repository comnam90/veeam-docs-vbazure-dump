# Creating Azure Files Backup Policies


To create a backup policy, do the following:

1. [Launch the Add Azure Files Policy wizard](fs_backup_wizard.md).
2. [Specify a backup policy name and description](fs_backup_name.md).
3. [Configure backup source settings](fs_backup_source_settings.md).
4. [Create a schedule for the backup policy](fs_backup_policy_schedule.md).
5. [Specify automatic retry settings and notification settings for the backup policy](fs_backup_retry_notifications.md).
6. [Review the estimated cost of protecting the selected Azure file shares](fs_backup_cost.md).
7. [Finish working with the wizard](fs_backup_finish.md).
