In this article

To add a new repository, do the following:

1. [Launch the Add External Repository wizard](repository_console_wizard.md).
2. [Specify an appliance, and provide repository name and description](repository_console_appliance.md).
3. [Configure repository settings](repository_console_storage_account.md).
4. [Specify a service account to access a blob container](repository_console_service_account.md).
5. [Select a blob container](repository_console_container.md).
6. [Enable data encryption](repository_console_encryption.md).
7. [Wait for the repository to be added to the backup infrastructure](repository_console_progress.md).
8. [Finish working with the wizard](repository_console_finish.md).
