In this article

To configure the virtual network configuration backup policy settings, perform the following steps:

1. [Launch the Virtual Network Configuration Backup wizard](vnet_backup_wizard.md).
2. [Select Azure subscriptions to protect](vnet_backup_subscriptions.md).
3. [Enable additional backup copy](vnet_backup_copies.md).
4. [Configure retention settings for Azure virtual network configuration backups](vnet_backup_retention.md).
5. [Specify automatic retry settings and notification settings](vnet_backup_notifications.md).
6. [Finish working with the wizard](vnet_backup_finish.md).
