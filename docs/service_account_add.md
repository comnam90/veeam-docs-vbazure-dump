# Adding Service Accounts


To add a new service account, do the following:

1. [Launch the Add Account wizard](service_account_wizard.md).
2. [Specify an account name and description](service_account_name.md).
3. [Choose an account type](service_account_type.md).
4. [Choose a scope for the account](service_account_scope.md).
5. [Specify account roles](service_account_select_roles.md).
6. [Check the required permissions](service_account_check_permissions.md).
7. [Finish working with the wizard](service_account_finish.md).
