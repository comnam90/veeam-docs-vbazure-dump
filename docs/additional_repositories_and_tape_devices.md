# Additional Repositories and Tape Devices


Additional repositories and tape devices are any repositories where Veeam Backup & Replication keeps and stores copies of Azure VM backups. For more information, see the Veeam Backup & Replication User Guide, sections [Backup Repository](https://helpcenter.veeam.com/docs/vbr/userguide/backup_repository.html?ver=13) and [Machines Backup to Tape](https://helpcenter.veeam.com/docs/vbr/userguide/backup_to_tape_jobs.html?ver=13).
