# Planning and Preparation


Before you start using Veeam Backup for Microsoft Azure, consider the following requirements:

* [Hardware and software requirements](system_requirements.md).
* [Network ports that must be open to ensure proper communication of Veeam Backup for Microsoft Azure components](ports.md).
* [Azure services to which Veeam Backup for Microsoft Azure must have outbound internet access](azure_services.md).
* [Permissions that must be assigned to accounts used to perform operations using the Veeam Backup & Replication console](plugin_permissions.md).
* [Permissions that must be assigned to service accounts used to perform Veeam Backup for Microsoft Azure operations](service_account_permissions.md).
* [Azure resource providers that must be registered in subscriptions](azure_providers.md).
* [Considerations and limitations that should be kept in mind before you deploy Veeam Backup for Microsoft Azure](limitations.md).
