In this article

To create a schedule-based backup policy, do the following:

1. [Launch the Add VM Policy wizard](vm_backup_wizard.md).
2. [Specify a backup policy name and description](vm_backup_name.md).
3. [Configure backup source settings](vm_backup_source_settings.md).
4. [Configure guest processing options](vm_guest_processing.md).
5. [Configure backup target settings](vm_backup_target_settings.md).
6. [Create a schedule for the backup policy](vm_backup_policy_schedule.md).
7. [Specify automatic retry, health check and notification settings for the backup policy](vm_backup_retry_notifications.md).
8. [Review the estimated cost of protecting the selected Azure VMs](vm_backup_cost.md).
9. [Finish working with the wizard](vm_backup_finish.md).
