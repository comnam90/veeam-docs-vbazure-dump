# Restoring Configuration Data Using Console


To restore the configuration database of a backup appliance using the Veeam Backup & Replication console, do the following:

1. [Check prerequisites and limitations](configuration_restore_console_limitations.md).
2. [Launch the Configuration Restore wizard](configuration_restore_console_wizard.md).
3. [Choose a backup file](configuration_restore_console_file.md).
4. [Review the backup file info](configuration_restore_console_info.md).
5. [Specify a decryption password](configuration_restore_console_password.md).
6. [Choose restore options](configuration_restore_console_options.md).
7. [Specify a user whose credentials will be used to connect to the appliance](configuration_restore_console_credentials.md).
8. [Wait for the restore process to complete](configuration_restore_console_progress.md).
9. [Finish working with the wizard](configuration_restore_console_finish.md).
